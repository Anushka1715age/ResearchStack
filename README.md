# ResearchStack

ResearchStack is a web application designed to help students and researchers manage their research workflow efficiently.

Features:
- Project-based research organization
- Research paper editor with templates (IEEE, APA)
- PDF storage per project
- Highlighting and note-taking inside research papers

Tech Stack:
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

Status:
MVP Development Phase
